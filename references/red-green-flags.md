# 面试红旗与绿旗速查表

## 🚩 红旗（扣分项）

### 数据意识
- 被问"这个数字好不好"时回避或绕开 → 数据敏感度差
- 说不出项目的核心指标 → 可能不是真正负责
- 把结果归因于单一因素 → 缺乏多因素思考

### 专业能力
- 项目描述只有"做了什么"没有"为什么这么做" → 执行者不是决策者
- 技术概念张冠李戴 → 基础不扎实
- 竞品分析停留在功能对比 → 缺乏深度思考

### 沟通
- 回答冗长抓不住重点 → 结构化表达弱
- 被追问时开始绕圈子 → 可能在掩饰
- 从不反问 → 缺乏好奇心或准备不足

### 诚信
- 简历写"负责"但追问下来是"参与" → 夸大贡献
- 时间线有明显断层不主动解释 → 可能隐瞒
- 项目数据前后矛盾 → 可能编造

### 态度
- 贬低前公司/团队 → 团队协作风险
- 只关心薪资和title → 动机可疑
- 对岗位和公司完全不了解 → 准备不足

## 🟢 绿旗（加分项）

### 思维能力
- 主动追问背景和约束条件 → 严谨
- 承认"不知道"然后说推理过程 → 诚实+思考力
- 能举一反三，从个例提炼方法论 → 抽象能力强

### 产品Sense
- 能从用户角度出发而不是技术角度 → 用户导向
- 关注"为什么不做"而不只是"做什么" → 有取舍思维
- 对竞品有自己的判断而不是复述功能列表 → 独立思考

### 数据驱动
- 主动提数据支撑论点 → 数据意识好
- 知道指标之间的关联和trade-off → 体系化思维
- 能质疑数据的合理性 → 不盲目相信数字

### 成长性
- 有明确的学习路径和最近在学什么 → 自驱
- 能反思自己项目的失败和教训 → 成熟度
- 对新领域展示好奇心而不是恐惧 → 适应力

### 校招特有加分
- 有高质量的实习经历（大厂+创业公司组合） → 视野广
- 能清楚说出"为什么选产品经理" → 职业认知清晰
- 课外项目/竞赛有实际产出 → 不只是刷简历
